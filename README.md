# Simple Redis Clone in Java

This is a lightweight Redis-like key-value store implemented in Java. The project aims to replicate basic Redis functionality as a learning exercise and for educational purposes.

## Features Implemented

- Basic key-value storage with `SET`, `GET`, `DEL`, `EXP` and `TTL` commands
- Single-client support
- Simple in-memory data storage

## Technologies Used

- Java (openjdk-24)
- Build Tool: Gradle Kotlin DSL (.kts)
- Core Java libraries (no external dependencies)

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Avik0928/Caching_server.git
   cd redis-clone-java
   ```
2. Go to:
   ```bash
   build\libs
   ```
   And run
   ```bash
   Redis_Clone-1.0-SNAPSHOT.jar
   ```
   On you machine
